# VibeKit Studio

VibeKit Studio is a simple page builder that lets you create, customize, and publish pages quickly.  
You can write content, preview it live, and share it using a public link.

---

## Features

- Signup and login with secure authentication
- Create, edit, and delete pages
- Live preview while editing
- Multiple themes (dark, light, minimal, etc.)
- Publish/unpublish pages with a shareable link
- Public page with view counter
- Dark/light mode toggle
- Toast notifications for actions

---

## Tech Stack

- Next.js (Pages Router)
- MongoDB with Mongoose
- JWT authentication
- bcryptjs for password hashing
- CSS Modules for styling
